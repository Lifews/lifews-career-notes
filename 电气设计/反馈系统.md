# 光栅尺

### 物理参数

1. 栅距
   - 定义：光栅尺上相邻两条刻线之间的物理距离。这是光栅尺的“物理刻度”。
   - 单位：微米、纳米。
   - 常见值：20µm， 10µm， 5µm， 1µm（高精度）。
   - 意义：是计算其他所有参数的基础。栅距越小，理论上可以达到的分辨率越高，但对制造工艺和信号处理的要求也越苛刻。

###  电气与信号参数

1. 信号周期

   - 定义：光栅尺扫描头输出一个完整电信号周期（如一个正弦波）所对应的物理位移量。在绝大多数情况下，信号周期等于栅距。
   - 意义：这是从物理尺度转换到电信号的基准。驱动器最初接收到的就是一个个信号周期。

2. 原始信号

   - 定义：扫描头直接输出的未经处理的模拟信号，通常是两路相位差90°的正弦/余弦信号（A， B）。
   - 意义：高质量的原始信号是获得高分辨率和高速度的基础。其质量用以下参数衡量：
     - 幅值：信号的电压强度，通常为1Vpp。
     - 正交性：A、B两路信号的相位差是否严格为90°。
     - 占空比：对于方波信号，高低电平的比例。
     - 谐波失真：信号偏离理想正弦波的程度。

3. 插补倍数 / 电子细分

   - 定义：驱动器内部的专用芯片（插补器）对原始正弦/余弦信号进行内插细分的倍数。

   - 如何工作：通过检测和计算A、B信号在一个周期内的微小电压变化，将一个物理信号周期“分割”成N份。

   - 常见值：5倍， 10倍， 20倍， 50倍， 100倍， 2048倍，甚至更高。

   - 核心公式：
     `系统分辨率 = 栅距 / 插补倍数`

     - 例如：栅距20µm， 200倍细分 -> 分辨率 = 20µm / 200 = 0.1µm。

   - 意义：这是在不改变物理栅距的情况下，大幅提升系统分辨率的关键技术。细分倍数越高，分辨率越高，但对信号质量和电路抗干扰能力要求也越高。

   - > [!CAUTION]
     >
     > 现代光栅尺规格书：
     >
     > - 不提供一个固定的“插补倍数”，因为这属于下游处理。
     > - 重点提供 栅距、精度 和 细分误差。
     > - 细分误差是评估其支持高倍细分能力的核心指标。误差越小，信号质量越高，能实现的有效细分倍数就越高、越稳定。

4. 输出信号类型

   - 增量式方波信号：将正弦波整形成方波后输出。最常见的是差分信号（A， B， Z）。
     - 线驱动输出：RS-422标准，抗干扰能力强，传输距离远。
   - 1Vpp正弦波信号：直接输出模拟正弦/余弦波，由驱动器的编码器接口进行细分。
   - EnDat， BiSS， FANUC， Mitsubishi等串行协议：输出的是包含绝对位置信息的数字数据包。这是目前高性能系统的绝对主流。
     - 优势：上电即知绝对位置，抗干扰极强，传输数据丰富（位置、报警、温度等），可靠性高。
   - 绝对位置值输出：如SSI协议，并行输出等。

###  精度与误差参数

1. 精度
   - 定义：在整个测量长度内，光栅尺指示值与实际绝对位置之间的最大偏差。这是光栅尺最重要的性能指标。
   - 单位：微米。
   - 表示方法：通常标注为 ±3µm， ±5µm/m（每米误差），或更高精度如 ±0.5µm。
   - 意义：决定了系统能实现多高的定位精度。精度由光栅尺的制造工艺（刻线精度、基材稳定性）保证，无法通过后续调试或电子细分提高。
2. 细分误差 / 插补误差
   - 定义：在一个信号周期内，由于原始正弦波信号质量不理想（如幅值不等、相位不正交、波形失真）以及电子细分电路的非线性，导致细分点不均匀而产生的误差。
   - 单位：通常为信号周期的百分比，如 < ±1%。
   - 意义：影响系统在微米/亚微米级运动时的平滑性和重复性。高质量的光栅尺和驱动器能提供极低的细分误差。
3. 重复精度
   - 定义：多次到达同一位置时，读数值的离散程度。通常远高于（优于）精度值。
   - 意义：决定了系统做重复运动的可靠性和一致性。

###  动态与安装参数

1. 最大运动速度
   - 定义：光栅尺能稳定输出正确信号的最高速度。受信号频率响应和接口协议限制。
   - 计算：与信号周期和输出频率有关。例如，栅距20µm， 输出频率1MHz， 则理论最大速度 = 20µm * 1,000,000/s = 20m/s。
   - 意义：必须满足您的电机最高速度需求。
2. 响应频率
   - 定义：扫描头能处理的信号最高频率。
   - 意义：决定了在高速下能否稳定工作。
3. 安装公差
   - 定义：扫描头与光栅尺之间的间隙、俯仰角、偏航角的允许误差范围。
   - 意义：安装是光栅尺可靠工作的关键！ 超出公差会导致信号衰减、丢数甚至损坏。必须严格参照手册使用专用安装工具。
4. 热膨胀系数
   - 定义：光栅尺基材（通常是玻璃、钢带、陶瓷）长度随温度变化的比率。
   - 单位：ppm/K（百万分之一每开尔文）。
   - 意义：对于大行程（>1米）和高精度系统至关重要。为了补偿温度影响，有几种方案：
     - 使用与被测对象相同材料的光栅尺（如钢带尺安装在钢制机台上）。
     - 使用低膨胀材料（如零膨胀玻璃、陶瓷）。
     - 外加温度传感器进行软件补偿。

### 其他

1. 参考零位 / 原点信号
   - 增量式系统：`Z`信号或`R`信号，提供一个每圈/每段一个的绝对参考点。
   - 绝对式系统：上电即有一个唯一的绝对坐标。
2. 防护等级
   - 定义：如IP53， IP67等，表示防尘防水能力。
   - 意义：决定光栅尺能否应用于有油污、冷却液或粉尘的工业环境。
3. MTBF（平均无故障时间）
   - 定义：可靠性指标。
   - 意义：对于高可靠性要求的自动化生产线非常重要。

### 有关调试

对于您当前调试的直线电机系统，请重点关注以下参数，并按此顺序检查：

1. 确认基础匹配：确保驱动器中设置的 “栅距”（或信号周期）与光栅尺本体上的铭牌一致。这是位置反馈的物理基准。
2. 验证接口与协议：在驱动器中选择正确的编码器类型（如EnDat， BiSS， 1Vpp Sine等），确保与扫描头匹配。
3. 检查安装质量：用塞尺等工具严格检查扫描头与光栅尺的间隙，确保在手册规定的公差内。安装不当是信号差、噪音大、丢数的首要原因。
4. **理解分辨率来源：在驱动器内看到的分辨率（如0.1µm），是 `栅距 ÷ 插补倍数` 计算出来的。**
5. 精度是硬指标：记住，光栅尺的精度（如±3µm）是出厂决定的，无法通过调试提高。如果您的应用要求±1µm的定位精度，那么必须选用精度高于±1µm的光栅尺。

### ABZ相

**A、B相**：两路频率相同、但存在固定相位差的方法脉冲。

- 功能：位移测量与方向判断
  1. 计数：驱动器通过计数A相（或B相）的脉冲数量，可以知道电机轴转过了多少角度。例如，一个2500线的编码器，电机转一圈会产生2500个A脉冲和2500个B脉冲。
  2. 辨向：A、B两相信号在相位上相差 1/4个周期（即90°电角度）。通过检测哪一相信号领先，驱动器就能判断电机的旋转方向。
     - 正转：通常定义为A相领先B相90°。
     - 反转：B相领先A相90°。

**Z相**：也称为“零点信号”、“索引信号”或“原点信号”。

- 旋转电机：对于标准的增量式旋转编码器，其Z相（是电机转子每完成360°机械旋转（完整一圈）时，在一个固定的、唯一的机械角度位置产生一个脉冲。（通常宽度与A/B相相同或更宽）。这个脉冲出现在电机圆周上唯一的一个物理位置。它的作用是提供一个每圈一次的绝对位置参考点。
- 直线电机：光栅尺的Z相信号，（雷尼绍为例子）由粘贴在尺身上的物理参考点（磁条或反射片）发出的。一个行程里通常不止有一个Z相信号。 您可以粘贴多个参考点，从而在一条光栅尺上获得多个Z相脉冲信号。但其中通常只有一个被选定为整台机器的坐标原点（Home点）。

为什么Z相至关重要？（与“寻相”直接相关）

1. 上电时的困境：系统刚通电时，驱动器通过A、B相知道电机转动的相对位移，但不知道电机轴当前位于圆周的哪个绝对角度。
2. Z相的作用：驱动器在电机旋转过程中，会持续寻找Z脉冲。一旦检测到Z脉冲，驱动器就知道了：“此刻，电机轴正精确地经过机械上的那个唯一零点位置”。
3. 建立映射：系统可以将这个Z脉冲出现瞬间的电气相位（如U相反电动势过零点）与机械位置进行对齐。从此以后，所有的A、B相计数都以这个Z点为基准进行累加，从而实现了机械位置与电气周期的同步。这就是“寻相”过程的物理基础之一。









# 直线电机

### 极距

直线电机的极距，是指沿直线电机运动方向（初级或次级长度方向）上，相邻两个同性磁极中心线之间的距离。

对应的是旋转电机在定子内圆上，一个N极中心到下一个N极中心（或一个S极中心到下一个S极中心）所对应的圆周弧长，当这个圆弧被“展开”成直线后，其长度就是直线电机的极距。

- 符号：通常用希腊字母 τ（Tau）表示。
- 单位：毫米（mm）或米（m）。

### **从测量到控制**

1. 极距是什么：如前所述，极距是电机本身磁场的一个完整周期的物理长度（例如，一个N极中心到下一个N极中心的距离）。对于电机驱动器来说，如果不使用光栅尺，它只能知道电机大致处于哪个磁极的“区域”，但无法知道在这个区域内具体的位置。
2. 光栅尺分辨率是什么：光栅尺是一个独立的、高精度的位置测量传感器。它的分辨率通常指 “每毫米的刻线数” 或直接就是能输出的最小位移单位（例如0.1 µm, 1 nm）。它直接、精确地测量动子的绝对或相对位置。
3. 两者如何协同工作：
   - 直线电机需要驱动器根据动子的精确位置来实时控制三相电流，以产生平稳、连续的推力。这称为“换相”。
   - 极距（τ）对应磁场一个完整周期的360°电角度。
   - 光栅尺的作用，就是告诉驱动器：“动子现在处于这个磁周期的哪一个精确的‘电角度’上”。这样驱动器才能施加匹配的电流。

> [!IMPORTANT]
>
> 位置反馈对于电角度的分辨率：在电机的一个电周期（360°电角度）内，光栅尺能提供多少个位置采样点
> $$
> 一个电周期内的采样点数=极距\div光栅尺的测量步长
> $$

例子：

- 直线电机极距 τ=30 mm*τ*=30 *mm*。

- 光栅尺的反馈分辨率（最小测量步长）为 0.1 µm=0.0001 mm0.1 µ*m*=0.0001 *mm*。

- 那么，在一个磁周期内，光栅尺可以提供：

  30 mm0.0001 mm/步=300,000 步0.0001 *mm*/步30 *mm*=300,000 步

  这相当于在一个360°电角度周期内，光栅尺可以分辨出 30万个不同的位置点。驱动器根据这些点来精确调整电流矢量，从而实现极平滑的控制，避免推力波动。

问题：

- 谁是决定性因素？ 整个直线电机系统的最终位置控制分辨率，根本上是由光栅尺的分辨率决定的。光栅尺能测到多准，系统理论上就能控制到多准。
- 公式的目的何在？ 公式 `极距 / 光栅尺分辨率` 的比值，用于评估 控制平滑度 和 换相精度。这个比值越大，意味着在电机的每个磁场周期内获得的位置信息越多，控制就越精细，电机运行越平稳，低速性能越好，推力纹波越小。
- 选型指导意义：在为一个给定的直线电机（极距τ已定）选择光栅尺时，为了保证高精度的控制，通常要求光栅尺的分辨率远小于极距。例如，一个30mm极距的电机，配套一个1µm分辨率的光栅尺是合理的；如果配套一个10µm的，控制平滑度就会差很多。









# 寻相

### **概念**

- **寻相**：伺服驱动器在启动或运行前的一种初始化过程，目的是找到电机的初始位置或相位角信息，确保电机和驱动器的控制保持同步。它的核心任务是确定电机转子的实际位置，以便驱动器能够正确生成控制信号（如电流或电压波形），从而精确控制电机。
- **机械角度**：指电机转子实际转过的物理角度（0° 到 360° 为一圈）。
- **电角度**：指电机内部定子旋转磁场的相位变化角度，或者说是驱动三相电流（UVW）所合成的矢量角度。它的变化速度与转子磁场的极对数（P*P*）相关：
  - 电角度=机械角度×极对数(P)电角度=机械角度×极对数(*P*)
  - 对于一个 **2极电机**（P=1），机械转一圈，电角度也变化360°。
  - 对于一个 **4极电机**（P=2），机械转一圈，电角度变化720°（即两个完整的电气周期）。

>
> 霍尔传感器：能把变化的磁场转换为输出电压的变化，即能检测是否存在磁场，以及磁场的强弱和方向。
>
> 在电机上电瞬间的快速、**粗略定位**转子位置。

| 特性         | **霍尔传感器**                      | **增量式编码器（ABZ）**                  |
| :----------- | :---------------------------------- | :--------------------------------------- |
| **信号类型** | **开关量**（高/低电平）             | **脉冲序列**（A/B相位差）                |
| **提供信息** | **粗略的绝对位置**（每周期6个扇区） | **精细的相对位移和方向**                 |
| **上电时**   | **立刻知道**转子在哪个60°扇区       | **不知道**绝对位置，是“蒙眼的”           |
| **精度**     | **低**（典型为60°电角度）           | **极高**（每转可达成千上万个脉冲）       |
| **核心任务** | **启动向导，解决“我在哪”**          | **运动导航，解决“我走了多少，要走多准”** |
